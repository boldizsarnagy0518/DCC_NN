Methodology note

This POC validates NN's actionable GEO recommendations through a controlled before/after mockup. The same prompts are tested without and with recommendation mockup assets, while model setup and scoring logic remain constant. Websearch is intentionally not used in the core POC so the impact of the proposed NN assets can be isolated.

Primary goal: recommendation impact validation Measures controlled answer quality, not live ranking Mockup assets represent proposed NN recommendations

What this POC validates

By keeping prompts and model labels constant, the dashboard isolates whether the proposed Q&A pages, calculators, decision guides, entity signals, original research and third-party proof improve answer quality.
